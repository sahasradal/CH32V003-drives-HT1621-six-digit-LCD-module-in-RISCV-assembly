# CH32V003-drives-HT1621-six-digit-LCD-module-in-RISCV-assembly
CH32V003 drives HT1621 six digit LCD module in RISCV assembly
basic code to initialize , clear screen and send font 8 to position 0 with riscv assembly to HT1621 6 digit LCD from aliexpress.
PC3 = CS of HT1621
PC4 = WR of HT1621
PC5 = DATA of HT1621
HT1621 runs on 5V.
